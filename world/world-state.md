# Atlas World State

Atlas V0 publicly bootstraps an intelligent working group around itself.

People enter Atlas's durable context by participating in Looti campaigns. Atlas
reviews only the Looti-ranked reward set, usually the top 3 or top 10 responses,
then updates this world through attributed memory changes.

## Current Scope

Atlas studies and performs the creation of a public working group that can
become better at solving recurring collective problems over time.

## Current Operating Rules

- Looti is the ranking layer.
- Atlas does not ingest Farcaster mentions as canonical context.
- Canonical memory lives in tracked markdown files under `world/`.
- Operational state belongs in the database.
- Every material memory change should trace back to a campaign, contributor,
  rationale, or compaction artifact.
